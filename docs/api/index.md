# API Reference

The `sofab` namespace is the runtime stream core: a streaming encoder
(<xref:sofab.OStream>) and a visitor-pattern decoder
(<xref:sofab.IStream> + <xref:sofab.IVisitor>),
plus the supporting types (<xref:sofab.FixlenType>,
<xref:sofab.ArrayKind>, <xref:sofab.SofabError>,
<xref:sofab.SofabException>, <xref:sofab.FlushSink>).
