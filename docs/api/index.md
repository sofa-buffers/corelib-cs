# API Reference

The `SofaBuffers` namespace is the runtime stream core: a streaming encoder
(<xref:SofaBuffers.OStream>) and a visitor-pattern decoder
(<xref:SofaBuffers.IStream> + <xref:SofaBuffers.IVisitor>),
plus the supporting types (<xref:SofaBuffers.FixlenType>,
<xref:SofaBuffers.ArrayKind>, <xref:SofaBuffers.SofabError>,
<xref:SofaBuffers.SofabException>, <xref:SofaBuffers.FlushSink>).
